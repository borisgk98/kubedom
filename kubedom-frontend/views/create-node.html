Добавление узла поставщика

Минимальные системные требования для узла:
ОЗУ 4 Гб
ЦПУ (кол-во) 2
Диск (свободное место) 20 Гб
Для того, чтобы добавить узел в сеть, нужно ввести следующую команду: 
sudo $(wget -O - https://www.kubedom.borisgk.space/static/install-provider-node-manager.sh | sudo sh -)
После этого нужно отредактировать конфиг: 
sudo nano /etc/kubedom/config.ini
Параметры конфига:
Параметр Описание
token Ваш токен поставщика, получить тут
node_uuid Случайный uuid для машины, можно получить с помощью команды uuidgen. Так же можно вставить токен уже существовавший ранее машины, если она перешла в статус DETACHED.
После того, как конфиг подправлен, нужно запустить менаджер следующей командой: 
sudo systemctl start kubedom-provider-node-manager.service
Так же рекомендуется добавить менаджер в автозагрузку: 
sudo systemctl enable kubedom-provider-node-manager.service
